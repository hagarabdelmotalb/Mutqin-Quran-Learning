تعليمات استخدام نموذج Tasmee3

هذه الواجهة تتصل بـ API موجود على: https://mahmoudgomaa8545-tasmee3-mutqin.hf.space/. النموذج يتوقع استقبال ملف صوتي مُرسَل كـ multipart/form-data تحت اسم الحقل: audio_file.

مُلخص الطلب

  • الرابط: https://mahmoudgomaa8545-tasmee3-mutqin.hf.space/
  • حقل الملف: audio_file (نوع: audio/wav أو أي ملف صوتي)

خطوات سريعة لاختبار الملف الصوتي

  1. سجّل أو ارفع ملف صوتي (WAV أو MP3) عبر واجهة التسجيل/رفع الموجودة أعلى الصفحة.
  2. اضغط زر إرسال/Submit لإرسال الملف إلى الـ API. سيتم إرسال الملف كـ audio_file.
  3. انتظر نتيجة المعالجة؛ ستعود الاستجابة (عادة JSON أو نص) وتُعرض في حقل النتائج.
  4. إن أردت تجربة خارج التطبيق، استخدم أمر الـ curl أعلاه.
